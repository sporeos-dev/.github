# Spore OS

Spore OS is a modular, distributed operating environment built around a socket-based hub-and-node architecture. This repository holds org-wide community and governance documents.

## Repositories

| Repository | Description |
| :--- | :--- |
| [spore-os-protocol](https://github.com/sporeos-dev/spore-os-protocol) | Protocol specification and manifest schema |
| [spore-os](https://github.com/sporeos-dev/spore-os) | Central hub implementation |
| [spore-core-nodes](https://github.com/sporeos-dev/spore-core-nodes) | Shell, REPL, observer, logging, and dialog nodes |
| [spore-client-libs](https://github.com/sporeos-dev/spore-client-libs) | Go reference implementation for node-to-hub communication |
| [spore-install](https://github.com/sporeos-dev/spore-install) | Build, packaging, and installation scripts |
